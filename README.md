# 🚀 Project Overview

**OnlineShop Django** is a modern web-based **E-Commerce platform** designed for product browsing, shopping cart management, secure ordering, and online store administration.

The project demonstrates:

✔ Full-Stack Web Development

✔ Django Backend Development

✔ Database Integration

✔ User Authentication

✔ Responsive UI Design

✔ Shopping Workflow Implementation

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/alipourfereydoon/onlineshop-django.git
```

## Move Into Project Directory

```bash
cd onlineshop-django
```

## Create Virtual Environment

```bash
python -m venv venv
```

## Activate Environment

### Windows

```bash
venv\Scripts\activate
```

### Linux / macOS

```bash
source venv/bin/activate
```

## Install Dependencies

```bash
pip install -r requirements.txt
```

## Apply Migrations

```bash
python manage.py migrate
```

## Run Development Server

```bash
python manage.py runserver
```

Open browser:

```bash
http://127.0.0.1:8000/
```

# 🚀 Deployment

You can deploy this project using:

### Render

```bash

```

### Railway

```bash

```

### VPS / Ubuntu Server

```bash
gunicorn projectname.wsgi:application
```
